# Storage Order Line Item

A Storage Order Line Item represents a specific product and its details within a larger Storage Order. It captures information about individual products being stored, including their dimensions and calculated space.
Creating a Storage Order Line Item

When creating a Storage Order Line Item, you'll need to specify:

- Storage Order ID: The ID of the parent Storage Order this line item belongs to.
- Product ID: The ID of the product being stored.
- Quantity Stored: The quantity of the product being stored.
- Manual Dimensions Length: The manually entered length of the product for storage calculation.
- Manual Dimensions Width: The manually entered width of the product for storage calculation.
- Manual Dimensions Height: The manually entered height of the product for storage calculation.
- Calculated Space for Item: The calculated space (e.g., cubic feet, square feet) that this specific item occupies.
- Entry Date: The date the product entered storage.
- Retrieval Date: The date the product was retrieved from storage.
