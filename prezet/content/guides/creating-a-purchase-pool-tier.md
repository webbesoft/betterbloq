# Purchase Pool Tier

Purchase Pool Tiers allow you to define different discount levels based on the volume achieved within a Purchase Pool.
Creating a Purchase Pool Tier

When creating a Purchase Pool Tier, you'll specify:

- Name: The name of the tier (e.g., "Bronze", "Silver", "Gold").
- Description: A brief description of what this tier offers.
- Minimum Volume: The minimum volume (e.g., quantity) required to qualify for this tier's discount.
- Maximum Volume: The maximum volume for this tier. Leave blank if there's no upper limit.
- Discount Percentage: The percentage discount applied to orders that reach this tier.
- Purchase Pool: The specific Purchase Pool this tier belongs to.
