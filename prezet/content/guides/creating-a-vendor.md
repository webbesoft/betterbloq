# Vendor

A Vendor represents a supplier or provider of products. You can manage their contact information, address, and key operational details like preparation time.
Creating a Vendor

To create a Vendor, you'll need to provide the following details:

- Name: The name of the vendor.
- Phone: The vendor's primary phone number.
- Address:
    - Address Line 1: The primary street address of the vendor.
    - Address Line 2: Optional secondary address information (e.g., suite number).
    - City: The city where the vendor is located.
    - State: The state where the vendor is located (currently defaults to "Texas").
    - Postal Code: The postal code for the vendor's address.
    - Country Code: The country where the vendor is located (currently defaults to "USA").
- Preparation Time (Days): The number of days the vendor needs to prepare an order after a purchase pool ends. This is an integer value, with a minimum of 0.
- User: The user responsible for managing this vendor.
