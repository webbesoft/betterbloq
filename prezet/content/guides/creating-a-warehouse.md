# Warehouse

A Warehouse is a physical location where products are stored. You can define its capacity, default storage pricing, supported conditions, and contact information.
Creating a Warehouse

To set up a Warehouse, you'll need to provide:

- Name: The name of the warehouse.
- Phone: The primary phone number for the warehouse.
- Address:
    - Address Line 1: The primary street address.
    - Address Line 2: Optional secondary address information (e.g., suite number).
    - City: The city where the warehouse is located.
    - State: The state where the warehouse is located (currently defaults to "Texas").
    - Postal Code: The postal code for the warehouse.
    - Country Code: The country where the warehouse is located (currently defaults to "USA").
- Capacity & Pricing:
    - Total Capacity: The total storage capacity of the warehouse in defined units (e.g., 10,000 sq ft).
    - Total Capacity Unit: The measurement unit for the warehouse's capacity (e.g., "sq ft", "cubic meters").
    - Available Capacity: The current available storage capacity. This can be updated automatically or manually maintained.
    - Default Storage Price Per Unit: The default price charged per unit of storage if no specific storage tier applies.
    - Default Storage Price Period: The default billing frequency for storage at this warehouse (e.g., "Per Day", "Per Week", "Per Month", "Per Quarter", "Per Year").
- Supported Storage Conditions: A list of storage conditions the warehouse can accommodate (e.g., "dry", "ventilated", "climate_controlled", "refrigerated"). Press enter to add each condition.
- User: The user responsible for managing this warehouse.
- Is Active: A toggle indicating whether the warehouse is currently active and available for use.
